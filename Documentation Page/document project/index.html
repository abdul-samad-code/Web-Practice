Introduction

JavaScript is a powerful, flexible, and lightweight programming language used for both client-side and server-side web development.

It allows you to add interactivity to your web pages, such as dynamic content updates, form validations, animations, and more.

Example inline code: const greeting = "Hello";

Another snippet: let counter = 0;

  • JavaScript can run directly in the browser.
    • What you should already know

    Before diving deep into JavaScript, you should be comfortable with the following:

  • Basic HTML and CSS knowledge
  • Using a text editor or IDE
  • Basic understanding of computer logic and variables

    • Basic concept: HTML & CSS

    JavaScript and Java

    Although JavaScript and Java share similar names, they are very different languages.

    Compare: console.log("JS") vs System.out.println("Java")

  • JavaScript is interpreted, Java is compiled.
  • JavaScript is dynamically typed, Java is statically typed.
    • Hello world

    The simplest program to get started with JavaScript is the "Hello World" example:

    console.log("Hello, world!");

  • This code prints "Hello, world!" in the browser console.
    • Variables

    Variables are containers used to store data values in JavaScript.

    Example:

    let name = "Samad";

  • Use let for variables that may change.
  • Use const for constants.
